# TerrGYM

#### kh정보교육원 팀 프로젝트
<p>
헬스장 회원/트레이너 관리 웹 사이트입니다. <br />
회원은 본인의 회원권을 확인하거나 구매할 수 있고, 헬스장 관리자는 트레이너와 회원의 정보를 조회, 수정할 수 있습니다.
</p>
